Sistema Inventario PMA

Sistema de control para las diferentes bodegas del PROGRAMA MUNDIAL DE ALIMENTOS

Información del Proyecto

Descripción

Sistema Inventario para la organizacion internacional de las naciones unidas, es un sistema creado para mejorar y agilizar los procesos de control de los articulos que la ONG poseé dentro de sus diferentes bodegas y digitalizar los procesos interno que se realizan a la hora de consultar los articulos en existencia.
El sistema consta con varias funcionalidades como son: Reporteria, Impresion de documentos internos, Control de Vencimiento de algunos articulos, mensajeria interna entre otras.

Tecnología de Desarrollo

  • Laravel como framework
  • PHP como lenguaje de programación
  • Plantilla prediseñada como front-end
  • AZURE como hosting
  • SQL SERVER como gestor de base de datos
  • Visual Studio Code como editor de texto
  • Git y GitHub Como controlador de versiones
