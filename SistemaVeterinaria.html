Información del Proyecto

Descripción

El sistema de veterinaria lleva el registro de los clientes que llevan a sus mascotas a la clinica de acuerdo al número de veces que lleguen a visitar, se verifica el DUI y se le otorga un descuento y el sistema presenta el número de visita que lleva en la clinica y el costo total

Tecnología de Desarrollo

  • Plantilla prediseñada como front-end
  • 0webhosting como hosting
  • Firebase como gestor de base de datos
  • Visual Studio Code como editor de texto
  • Git y GitHub Como controlador de versiones
